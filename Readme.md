# MatroskaSharp

A refactoring and adaptation to PCL of [Olegzee's NEBml](https://github.com/OlegZee/NEbml).
Maybe not needed anymore.

## Supported platforms

- Windows 8.0+ (WinRT and UWP)
- Windows Phone 8.1+ (WinRT and UWP)
- .NET Framework 4.5+
- ASP.NET Core 1.0
- Xamarin
